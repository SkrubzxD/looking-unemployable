I've actually used Github before, so this would be really quick, let's recap

```
# Initializes a repo
git init

# Stages item
git add

# Commits all with a message to the local repo
git commit -ma

# Commits when a person with the same parent comit commits before you
git commit --rebase

# Push the local code to remote (maybe to BitBucket!)
git push

# Clones the repo
git clone 
# git branch makes new branch for isolating development.
git branch

# 
git pull = git fetch + git merge
```
